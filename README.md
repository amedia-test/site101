# Site101 test app

The purpose of this app is to be a simple playground app for adding SiteConfig to a node app.

## Requirements

- Node
- Npm

## Install

```bash
npm install
```

## Run

```bash
npm run build
npm run start
```

You can now open the app in your browser at [http://[::]:8080/api/site101/v1](http://[::]:8080/api/site101/v1)

## Adding site-config support

Doc: [https://github.com/amedia/node-site-config](https://github.com/amedia/node-site-config)


You can cheat by looking at the `siteconfig` branch in this repo.
